Managing Views

  • You can toggle views using the #hash value
  • You can toggle multiple views (separate with a plus sign)
  • Add as many as you need!

Export Views

- Use the **render** script from the terminal to export all screenshots - Update the **states** element to identify which views to render - **2x** will save Retina ready screenshots - **format** accepts PDF, but they aren't rendered as long scrolling pages
